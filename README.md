# FUTURE_DS_02
# Customer Retention & Churn Analysis Dashboard
# Overview
This project presents a Customer Retention & Churn Analysis Dashboard developed in Power BI. The dashboard helps analyze customer behavior, identify churn patterns, and evaluate factors influencing customer retention through interactive visualizations and key performance indicators (KPIs).
# Key Features
- Total Customers, Active Customers, and Churned Customers KPIs
- Churn Rate and Retention Rate Analysis
- Customer Churn Distribution
- Churn Analysis by Contract Type, Internet Service, Gender, and Payment Method
- Churn Rate Analysis by Tenure Group
- Interactive Filters for enhanced data exploration
# Tools Used
- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Data Visualization
- Business Intelligence & Analytics
# Objective
The goal of this project is to uncover customer churn trends, understand retention drivers, and provide actionable insights that support data-driven business decisions.

# Key Insights
- Customers with month-to-month contracts have the highest churn rate.
- Churn decreases as customer tenure increases.
- Fiber optic internet users show higher churn compared to other service types.
- Payment methods and contract types significantly influence customer retention.
- This project demonstrates skills in data analysis, dashboard design, KPI development, and business intelligence reporting using Power BI.

# Outcome
This dashboard helps businesses understand customer churn patterns, identify retention challenges, and make data-driven decisions to improve customer loyalty and reduce attrition.
